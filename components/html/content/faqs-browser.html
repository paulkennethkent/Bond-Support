Kerboodle Support Browser FAQs


What versions of browsers are supported for use with Kerboodle?

Please see our System Requirements document for details.

I’ve just gone to Kerboodle and there is a message that says I’m using an unsupported browser. What should I do?

If you have another browser supported for use with Kerboodle installed on your machine we suggest you switch browsers as the quickest solution. See What versions of browsers are supported for use with Kerboodle?, above, for a list of supported browsers. If that isn’t possible, please contact your school’s IT technician, who should be able to advise on whether it’s possible to update your browser or install a different supported one on your machine.

If you are using your home computer and want to update your browser, see I am using my home computer. Where can I get a new browser? below

You can continue to use Kerboodle in an unsupported browser, but please be aware that a large part of the content will not work (particularly interactive content, audio and video) and that much of the functionality may be slow or perform in unusual ways.

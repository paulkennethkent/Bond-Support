Kerboodle Support Filtering FAQs


Which email addresses do I need to whitelist and why?”

Please add the email addresses no-reply@kerboodle.com and support@kerboodle.com to your school’s accepted list of emails.

Every Teacher and Adminstrator will need to activate their accounts before using Kerboodle. If the account is not activated, it cannot be accessed. To do this, you will need to click on a link that we send to you from no-reply@kerboodle.com. Please whitelist this email address to make sure you receive your activation email.

If you contact us with any questions, you will receive an email from support@kerboodle.com. Adding this email to your accepted list of emails will ensure that you receive our response.

We realise that some emails can be interpreted as spam, so please check your Junk folders for this email before contacting us.

How and why do I need to whitelist certain web URLs?

We understand that schools need a high level of security when it comes to internet use within a school. Unfortunately some of these security features can sometimes have an effect on the loading of some Kerboodle pages and content. To avoid any potential issues like this you can add the Kerboodle domains and URLs to the school’s web filtering software to exclude them from the filtering process. We try to keep our list as up to date as possible. Please email us at support@kerboodle.com and ask for the latest list. If the problems still persist after adding them then please contact your Web Filtering provider to ensure the settings are applied correctly.

You can run your own tests with the developer tools within certain browsers to see which URLs are being blocked. If you want to find out more information about these developer tools then please click on the links provided below.
